# 快速启动指南

## 安装依赖

```bash
npm install
```

## 启动开发服务器

```bash
npm run dev
```

服务器将在 http://localhost:3000 启动

## 功能演示

### 首页功能
- 搜索框：输入关键词进行搜索
- 分类标签：点击不同分类进行筛选
- 热门推荐：点击推荐项目直接搜索

### 搜索页功能
- 实时搜索：输入关键词后自动搜索（防抖）
- 分类筛选：下拉选择资源类型
- 排序方式：按相关度、时间、大小排序
- 加载状态：搜索时显示骨架屏
- 分页功能：点击上一页/下一页

### 资源求助页功能
- 表单填写：资源名称、类型、描述、联系方式
- 表单验证：必填项验证和格式验证
- 提交功能：节流提交防止重复提交
- 成功提示：提交成功后显示确认信息

## 技术特点

### 响应式设计
- 桌面端：完整功能布局
- 平板端：自适应布局调整
- 移动端：垂直堆叠布局

### 性能优化
- 防抖搜索：500ms延迟减少请求
- 节流提交：1秒间隔防止重复提交
- 骨架屏：改善加载体验

### UI设计
- 简洁配色：蓝色渐变 + 白色卡片
- 平滑动画：悬停和点击效果
- 专业图标：Emoji图标系统
